Paper Calendar

Sexy Minimalistic Foldable Paper Art

What is this?

Back in the days everybody had paper calendars. Now they are back - In a sexy foldable format, for free and in various languages. A paper calendar is simplistic, minimalistic and fits in your pocket. Simplicity counts. Back to the roots with a subtle design. Some call it classic, some call it vintage, but it is definetly avant-garde.

About

My name is Jendrik, in my spare time I like to code social mini projects. see my website. My brother Philip wrote a Python tool called papr. The script output is a PDF file, a calendar for the next two months - foldable, so it will fit in your pocket. It is dead simple, but extremly cool. Unfortunately, most of the people do not know how to use a Python terminal tool, so I set up this micro site.

This project is based on papr, Python, Cairo, Pango, Flask and runs on an Ubuntu Server.
