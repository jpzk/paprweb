Sexy Minimalistic Foldable Paper Art

What is this?

My brother wrote a Python script. The scripts outputs a PDF file, a calendar for the next two months - foldable, so it will fit in your pocket. It is dead simple, but extremly cool. Unfortunately, most of the people do not know how to use a Python terminal tool, so I set up this micro site. Go to print it and download your calendar for free.
